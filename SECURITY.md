# Security Policy

## Reporting a Vulnerability
Found an insecure default or misconfiguration in the modules/examples?

1. **Do not** open a public issue.
2. Use GitHub **[Private vulnerability reporting](https://github.com/devsecforge/secure-terraform-aws/security/advisories/new)** (Security tab → "Report a vulnerability").
3. Include the file, the issue, and its impact.

Acknowledgement within **72 hours**.

## Note on the insecure example
`examples/insecure/` is **intentionally insecure** for training and is excluded from gating scans.
Its issues are documented, not vulnerabilities to report.
